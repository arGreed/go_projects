Проект:
    Консольное приложение "Todo List".

Задание:
    Реализовать простую систему для управления списком дел.

Описание:
    Консольное приложение, в котором реализованы CRUD-команды для списка задач.
    Сам список хранится в json-файле.
    
Список команд:
    1) add - добавление задачи.
    2) delete - удаление задачи.
    3) update - изменение задачи.
    4) show - показать конкретную задачу по её идентификатору.
    5) all - показать список всех задач.
    6) exit - выход из приложения.
